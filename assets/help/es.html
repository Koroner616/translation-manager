Gestor de Traducciones - Ayuda

Primeros Pasos

El Gestor de Traducciones es una herramienta para administrar archivos de traducción multilingüe en formato JSON.

Trabajando con Traducciones

  1. Selecciona una carpeta que contenga tus archivos de traducción JSON utilizando el botón "Seleccionar Carpeta".
  2. Cada archivo JSON debe representar un idioma (por ejemplo, es.json, en.json).
  3. Edita las traducciones como se describe en la sección "Editando Traducciones".
  4. Haz clic en "Guardar Cambios" para guardar tus modificaciones.

Editando Traducciones

Tienes dos formas diferentes de editar traducciones:

  • Edición individual: Haz clic en cualquier campo de traducción para abrir un popup de edición con más espacio. Usa Ctrl+Enter para guardar o Escape para cancelar.
  • Edición múltiple: Haz clic en la clave de traducción (columna izquierda) para abrir un editor que te permite modificar todos los idiomas a la vez para esa clave.

Añadir Nuevos Idiomas

Para añadir un nuevo idioma:

  1. Ingresa el código del idioma en el campo en la parte inferior izquierda (ej. it, de).
  2. Haz clic en "Añadir Idioma".
  3. El nuevo idioma se creará con la misma estructura que los idiomas existentes, pero con valores vacíos.

Añadir Nuevas Traducciones

Para añadir una nueva clave de traducción:

  1. Haz clic en "Añadir Nueva Traducción".
  2. Selecciona dónde añadir la clave (en qué sección).
  3. Ingresa el nombre de la clave.
  4. Ingresa las traducciones para cada idioma.
  5. Haz clic en "Añadir Traducción".

Atajo: También puedes hacer clic en el botón "Añadir Clave" dentro de cualquier grupo para añadir directamente una traducción a ese grupo.

Buscando Traducciones

Utiliza el cuadro de búsqueda para filtrar traducciones por nombre de clave.

Consejo: Haz clic en los nombres de sección para expandirlos o contraerlos.

Carpetas Recientes

La aplicación recuerda tus carpetas abiertas recientemente. Haz clic en "Carpetas Recientes" para acceder rápidamente a ellas.

Atajos de Teclado

  • Ctrl+Enter (o Cmd+Enter en Mac): Guardar cambios en el popup de edición
  • Escape: Cerrar el popup de edición sin guardar
