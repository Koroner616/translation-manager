Translation Manager - Help

Getting Started

Translation Manager is a tool to manage multilingual translation files in JSON format.

Working with Translations

  1. Select a folder containing your JSON translation files using the "Select Folder" button.
  2. Each JSON file should represent one language (e.g., en.json, es.json).
  3. Edit translations as described in the "Editing Translations" section.
  4. Click "Save Changes" to save your modifications.

Editing Translations

You have two different ways to edit translations:

  • Individual editing: Click on any translation field to open an editing popup with more space. Use Ctrl+Enter to save or Escape to cancel.
  • Multiple editing: Click on the translation key (left column) to open an editor that allows you to modify all languages at once for that key.

Adding New Languages

To add a new language:

  1. Enter the language code in the input at the bottom left (e.g., it, de).
  2. Click "Add Language".
  3. The new language will be created with the same structure as existing languages, but with empty values.

Adding New Translations

To add a new translation key:

  1. Click "Add New Translation".
  2. Select where to add the key (in which section).
  3. Enter the key name.
  4. Enter translations for each language.
  5. Click "Add Translation".

Shortcut: You can also click the "Add Key" button inside any group to directly add a translation to that group.

Searching Translations

Use the search box to filter translations by key name.

Tip: Click on section names to expand or collapse them.

Recent Folders

The application remembers your recently opened folders. Click "Recent Folders" to quickly access them.

Keyboard Shortcuts

  • Ctrl+Enter (or Cmd+Enter on Mac): Save changes in the editing popup
  • Escape: Close the editing popup without saving
